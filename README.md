# Nest.js TDD tarter kit

This kit contains the folowing features

- Test setup
- Mongodb integration

# Installation

```bash
$ npm install
```

# Run

```bash
$ npm run start:dev
```

# Run with docker

```bash
$ docker-compose up build
```

# Accessing swagger documentation

[http://localhost:3000/docs/](http://localhost:3000/docs/)
